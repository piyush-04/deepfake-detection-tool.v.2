# OmniForensics-Stream

A streaming-first, zero-storage multimodal forensic detection system.


## Key ideas

- No stored datasets
- Real-time streaming
- On-the-fly fake generation
- Frozen pretrained encoders
- Lightweight forensic heads


Supported modalities:

- Text
- Audio
- Image
- Video (temporal aggregation)


## Usage

Train per modality, then run `app.py`.
